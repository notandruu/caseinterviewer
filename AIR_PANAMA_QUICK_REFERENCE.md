# Air Panama Case - Quick Evaluation Reference Card

## 📊 Case Facts (Memorize These)
```
Aircraft: 200 seats
Mango:    60% full × $500 = $60k/flight  (cancel 2)
NYC:      80% full × $700 = $112k/flight (add 3)
```

## 🎯 Ground Truth Math
```
Mango daily:  2 × $60k  = $120k
NYC daily:    3 × $112k = $336k
Net change:   +$216k/day
Annual:       +$79M/year
```
